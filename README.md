# 🚀 Feature Engineering Playbook: From Raw Data to ML Model

> Transforming raw data into powerful features that drive better machine learning performance.

---

## 📌 Overview

This repository is a complete, end-to-end guide to **Feature Engineering**, covering all essential techniques used in real-world machine learning projects.

It demonstrates how raw, unstructured data can be systematically transformed into meaningful features to improve model performance.

---

## 🎯 Objectives

* Understand the importance of feature engineering in ML
* Apply various techniques on real-world datasets
* Compare model performance before and after feature engineering
* Build a strong foundation for data science interviews

## 🔥 Techniques Covered

### 🧩 Missing Value Handling

* Mean / Median Imputation
* Mode Imputation
* Random Sample Imputation
* Constant Value Imputation

### 🔤 Categorical Encoding

* One Hot Encoding
* Label Encoding
* Frequency Encoding
* Target Encoding

### 📏 Feature Scaling

* Standardization
* Min-Max Scaling
* Robust Scaling

### 📊 Outlier Treatment

* IQR Method
* Z-Score Method
* Winsorization (Capping)

### 🔄 Feature Transformation

* Log Transformation
* Square Root Transformation
* Box-Cox Transformation

### 📅 Date-Time Feature Engineering

* Extract Year, Month, Day
* Weekday Extraction
* Time Differences

### 🧠 Feature Selection

* Correlation Analysis
* Chi-Square Test
* Mutual Information
* Tree-Based Feature Importance
